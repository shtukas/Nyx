
The Nyx Network is made of Nodes. They are Marbles on disk.

Node (Common)
    - uuid            : NodeId
    - unixtime        : Integer
    - nxType          : "NxTag" | "Url" | "Text" | "UniqueFile" | "StdFSTree" | "FSUniqueString"
                        "NxFLog"
    - description     : String

NxTag:
    - nxType : "NxTag"

Url:
    - nxType : "Url"
    - url    : String

Text:
    - nxType : "Text"
    - text   : String # The actual text

UniqueFile:
    - nxType : "UniqueFile"
    - nhash  : String # AionPoint (one file)

StdFSTree:
    - nxType : "StdFSTree"

FSUniqueString:
    - nxType       : "FSUniqueString"
    - uniquestring : String

NxFLog: 
"""
A NxFLog is a FSUniqueString, to refer to the root of the log, and each sub directory is uniquestring folder. If a child is added to the root in nyx, then that child has to be a unique string. If a folder is manually added to the folder then  new node and an arrow are added to Nyx.
"""
    - nxType       : "NxFLog"
    - uniquestring : String
